# 🔌 Laravel API Masterclass - Laracasts Project

This repository contains the RESTful API I built while completing the advanced **"Laravel API Master Class"** on [Laracasts](https://laracasts.com/series/laravel-api-master-class).

I built this project to deepen my knowledge of backend architecture, secure token-based authentication, and proper API design patterns in Laravel.

## ⚠️ Disclaimer & Credit
The course structure, concepts, and original architecture are provided by **Laracasts**. This repository serves as a practical showcase of my continuous learning and my ability to build robust, scalable APIs following modern industry standards.

## 🧠 Core API Concepts Mastered
Through this series, I leveled up my backend skills and successfully implemented:

* **RESTful Architecture:** Designing clean, predictable API routes and controllers.
* **Laravel API Resources:** Using Resources and Resource Collections to properly format and transform JSON responses, keeping the data presentation layer separate from the database structure.
* **Token-Based Authentication:** Securing endpoints and generating API tokens using **Laravel Sanctum**.
* **Advanced Querying:** Implementing dynamic filtering, sorting, and pagination via query string parameters directly into Eloquent.
* **Validation & Error Handling:** Using Form Requests for strict incoming data validation and ensuring consistent JSON error responses.

## 🛠 Tech Stack & Tools
* **Framework:** Laravel 10/11 (PHP 8+)
* **Authentication:** Laravel Sanctum
* **Database:** MySQL / SQLite
* **Testing / Interacting:** Postman / Insomnia
